# lexilegi-api
LexiLégi est une méthode de structuration, vérification et contre-analyse juridique.Les API juridiques sont pour consulter, sourcer, vérifier les textes, jurisprudences et références officielles. LexiLégi ne remplace pas un avocat. LexiLégi ne prend pas de décision juridique automatique. Aucun dossier réel n’est traité.
